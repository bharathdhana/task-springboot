# task-springboot
## A single project that covers Spring Boot, REST API design, CRUD, DTO, Validation, Exception Handling, JPA / Hibernate, and MySQL effectively. Practical project that demonstrates all thru TODO Sysatem

## Goal
backend service where users can perform Crud Operation on Tasks

## Stack
- Spring boot
- Spring Data JPA
- Hibernate
- MySQL
- Postman

## Features
- User Management
- Task Management

## DB Design
- User Table
- Task Table

# REST API Design
- User APIs
POST   /api/users
GET    /api/users
GET    /api/users/{id}


- Task APIs
POST   /api/tasks
GET    /api/tasks
GET    /api/tasks/{id}
PUT    /api/tasks/{id}
DELETE /api/tasks/{id}
